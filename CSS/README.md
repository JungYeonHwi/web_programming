# CSS

#### 기초 ####

* CSS란?
  + Cascading Style Sheets
  + 과거에는 디자인 정보를 HTML에 안에 포함 -> HTML 본연의 목적 (구조화된 문서) 상실
  + 디자인적 요소를 HTML과 완전히 분리시키기 위해 CSS 생성

* CSS을 HTML에 적용하는 방법
  + 인라인 스타일 : 태그 안에 적용
  + 내부 스타일 : HTML 안에 style 태그를 이용하여 적용
  + 외부 스타일 : CSS 파일 생성하여 HTML에 적용
